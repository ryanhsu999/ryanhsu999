Digital 1 Photography html UI/UX - How it was made
This website was made using HTML due to the following: I want to learn something new for my skill set and it's free to host a website on Github.
I believe that my website helps people learn about my work and it's good for future employers.
The overall process starts with the portfolio interface: Prototyping the first idea on paper. Later it was rejected due to not looking good and being too complex to implement in the time of one week.
So what I came up with another prototype. The advantage was that the pages look clear and more organized. And there is a nav that shows you what page you are on.
Thank you
VS code is used to write the HTML and Github is the host server. Using bootstrap and W3 School was a game changer because of the resourceful information on HTML.
This website wouldn't be possible without the following people and tools:
Digital 1 photography Photography Instructor Mr. Thomas Bartholomew-Couts for helping me to learn and grow.
Github, Bootstrap, VS code
Thank you again for visiting my website.
